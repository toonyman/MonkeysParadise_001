🎲 로또 번호 생성기 사용 가이드

프로 로또 번호 생성기는 완전한 랜덤 알고리즘을 사용하여 공정하고 예측 불가능한 번호를 생성합니다. 모든 번호는 동일한 확률로 선택되며, 과거 당첨 번호나 패턴과 무관하게 독립적으로 생성됩니다.

주요 기능

  • 다중 게임 생성: 한 번에 최대 10게임까지 생성 가능
  • 번호 제외 기능: 원하지 않는 번호를 미리 제외할 수 있음
  • 생성 이력 관리: 이전에 생성한 번호들을 확인 가능
  • 통계 추적: 총 생성 횟수와 오늘 생성 횟수 추적

로또 6/45 안내
